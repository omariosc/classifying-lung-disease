Training Your Own AI Model

This guide will walk you through the process of training your own AI model using Teachable Machine. This user‐friendly platform allows you to create machine learning models without any coding experience.

Step 1: Download This Repository

To get started, download this repository by clicking the green "Code" button and selecting "Download ZIP". Once downloaded, extract the contents of the ZIP file. Your folder structure should resemble the following:

|-- Exported Model
│   ├── keras_model.h5
│   ├── labels.txt
├── README.md
├── TEST
│   ├── healthy
│   │   ├── ...
│   ├── disease
│   │   ├── ...
├── TRAIN
│   ├── healthy
│   │   ├── ...
│   ├── disease
│   │   ├── ...
├── tutorial
│   ├── ...

The README.md file contains this tutorial and the tutorial folder holds the images that will assist you.

Step 2: Get Started with Teachable Machine

Visit the Teachable Machine website and click on "Get Started".

Select the "Image Project" option, then choose "Standard Image Model" to create an image classification model.

Step 3: Add Your Data

Rename your classes to "Healthy" and "Diseased" by clicking on the default text. Then, upload all the images from the repository (located in the healthy and disease folders within the TRAIN directory). Please note that if you are uploading many images, it may take some time.

Optionally, click on "Advanced" to adjust the training parameters.

Once all images are uploaded, click the "Train Model" button. It is advisable not to change tabs during this process.

Step 4: Test Your Model

After training, you can dismiss the camera prompt, switch the input mode to "File", and test your model using the images in the TEST folder.

Step 5: Stretch Exercises

  1. Try testing your model using the images in the Extension folder, which include samples for “bronchitis”, “cancer”, “cartoon”, “child” and “inverted” lungs, or use images from online sources.
  2. Use the exported model in either JavaScript or Python to validate all the images in the TEST folder from the full dataset available on Kaggle. Research how a confusion matrix can be interpreted.

Conclusion

Congratulations! You have successfully trained your own AI model to classify pneumonia in x‐ray images of lungs using Teachable Machine.

Export Model (Optional)

If you wish to explore further, you can export your trained model and integrate it with JavaScript or Python. A sample exported model is available in the Exported Model folder for guidance.
